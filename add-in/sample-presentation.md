# מצגת דוגמה לחידון Kahoot

## הוראות ליצירת מצגת עם ה-Add-in

כדי להשתמש ב-Add-in, צור מצגת PowerPoint עם התוכן הבא:

### שקף 1 - דף פתיחה
```
🎯 ברוכים הבאים לחידון הטריוויה!

משתתפים: !#users!
זמן נותר: !time! שניות

לחצו על "התחל משחק" כדי להתחיל
```

### שקף 2 - שאלה ראשונה
```
שאלה 1

מהי בירת ישראל?

א) תל אביב
ב) ירושלים  
ג) חיפה
ד) באר שבע

משתתפים פעילים: !#users!
זמן לתשובה: !time!
```

### שקף 3 - תשובה נכונה
```
✅ תשובה נכונה!

הבירת ישראל היא ירושלים

משתתפים: !#users!
```

### שקף 4 - שאלה שנייה
```
שאלה 2

באיזו שנה הוקמה מדינת ישראל?

א) 1947
ב) 1948
ג) 1949
ד) 1950

משתתפים פעילים: !#users!
זמן לתשובה: !time!
```

### שקף 5 - סיכום
```
🏆 סיים את החידון!

תודה לכל המשתתפים!
סה"כ משתתפים: !#users!

🎉 כל הכבוד!
```

## הוראות שימוש:

1. צור מצגת PowerPoint חדשה
2. הוסף את התוכן לעיל לשקפים
3. ודא שהמחרוזות `!#users!` ו-`!time!` מופיעות במקומות הרצויים
4. שמור את המצגת
5. פתח את ה-Add-in מהרצועה
6. לחץ "התחל משחק"
7. השתמש ב"עמוד הבא" כדי להתקדם

המחרוזות יוחלפו אוטומטיט בנתונים אמיתיים מהשרת!
