# Kahoot Quiz Manager - Office Add-in

Add-in ל-PowerPoint לניהול חידון טריוויה עם שרת מרכזי.

## תכונות

- 🚀 **אתחול משחק**: קריאה אוטומטית ל-API בהתחלת המצגת
- ➡️ **מעבר עמודים**: שליחת פקודות לשרת בכל מעבר עמוד
- 👥 **עדכון משתתפים בזמן אמת**: החלפה אוטומטית של `!#users!` במספר המשתתפים הנוכחי דרך WebSocket
- ⏰ **עדכון זמן בזמן אמת**: החלפה אוטומטית של `!time!` בזמן הנותר דרך WebSocket
- 🔄 **עדכון אוטומטי**: רענון נתונים אוטומטי באמצעות WebSocket
- 📊 **ממשק ניהול**: פאנל בקרה נוח בעברית
- 🏷️ **מערכת תיוג מתקדמת**: placeholders מתעדכנים ללא הגבלה

## דרישות מערכת

- Microsoft PowerPoint (Office 365 או Office 2019)
- חיבור לאינטרנט
- Node.js (לפיתוח)

## התקנה

### 1. הכנת הסביבה

```bash
# התקנת תלויות
npm install
```

**📋 דרישות:**
- Node.js 14+ מותקן מ-[nodejs.org](https://nodejs.org)

### 2. הפעלת שרת הפיתוח

```bash
npm start
```

השרת יתחיל על פורט 3000: `http://localhost:3000`

**💡 טיפ:** השרת משתמש ב-`http-server` פשוט ללא HTTPS

### 3. טעינת ה-Add-in ב-PowerPoint

#### אופציה א': Sideloading אוטומטי
```bash
npm run sideload
```

#### אופציה ב': טעינה ידנית
1. פתח PowerPoint
2. עבור ל: **Insert** → **My Add-ins** → **Shared Folder**
3. בחר בקובץ `manifest.xml`
4. לחץ **Add**

## שימוש

### 1. הכנת המצגת

יש לך שתי דרכים להוסיף מידע דינמי לשקף:

#### דרך A: שימוש בכפתורי הסרט (מומלץ! 🌟)
1. פתח את המצגת ב-PowerPoint
2. עבור לסרט **Kahoot Controls**
3. לחץ על:
   - **👥 Add Users Counter** - יוסיף תיבת טקסט עם "Participants: 0"
   - **⏰ Add Timer Display** - יוסיף תיבת טקסט עם "Time: 0s"
4. הזז את התיבות למקום הרצוי בשקף
5. התיבות כבר מתויגות ומוכנות לעדכונים אוטומטיים!

**יתרונות:**
✅ אין צורך לזכור מחרוזות מיוחדות  
✅ תיבות מוכנות ומתויגות מראש  
✅ עדכון מיידי עם נתונים קיימים  
✅ ניתן להזיז ולעצב בחופשיות  

#### דרך B: כתיבה ידנית של placeholders
בכל שקף שתרצה להציג מידע דינמי, הוסף:
- `!#users!` - יוחלף במספר המשתתפים הנוכחי
- `!time!` - יוחלף בזמן הנותר

לדוגמה:
```
ברוכים הבאים לחידון!
משתתפים: !#users!
זמן נותר: !time! שניות
```

**💡 איך זה עובד:**
- **דרך A**: הכפתורים יוצרים תיבות טקסט מתויגות מראש, מוכנות לעדכונים
- **דרך B**: בפעם הראשונה, המערכת מחפשת את המחרוזות `!#users!` ו-`!time!`, מחליפה אותן ומתייגת את הצורות
- בעדכונים הבאים, המערכת מחפשת את הצורות המתויגות ומעדכנת את התוכן שלהן
- **שתי הדרכים מאפשרות עדכונים אינסופיים בזמן אמת דרך WebSocket!**

### 2. הפעלת המשחק

1. פתח את המצגת ב-PowerPoint
2. אם השתמשת בדרך A, התיבות כבר מוכנות!
3. לחץ על **Control Panel** ברצועת הכלים **Kahoot Controls**
4. בפאנל שנפתח, לחץ **התחל משחק**
5. השתמש ב**עמוד הבא** כדי להתקדם במצגת
6. צפה בעדכונים בזמן אמת!

### 3. תכונות נוספות

- **עדכון אוטומטי**: הפעל לעדכון נתונים רציף
- **רענן נתונים**: עדכון ידני של הנתונים
- **מעקב סטטוס**: ראה מספר משתתפים וזמן נותר

## API Endpoints

ה-Add-in מתחבר לשרת בכתובת: `https://din-online.co.il/kahoot/`

- `?init` - אתחול משחק
- `?next_page` - מעבר לעמוד הבא
- `?get_users` - קבלת מספר משתתפים
- `?get_time` - קבלת זמן נותר
