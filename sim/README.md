# 🎮 Kahoot Simulator - סימולטור משתתפים

אפליקציית React לסימולציה של משתתפים במשחק Kahoot.

---

## 🎯 תכונות

- ✅ **10 משתתפים פיקטיביים** - כל אחד עם ID ושם
- ✅ **חיבור/ניתוק** - כפתור לכל משתתף
- ✅ **REST API** - קריאות לשרת הקיים
- ✅ **WebSocket** - עדכונים בזמן אמת
- ✅ **ממשק מעוצב** - עם אנימציות ומצבים ויזואליים

---

## 📦 התקנה

### 1. התקן תלויות
```bash
cd sim
npm install
```

### 2. הגדר את כתובת השרת

#### לשימוש מקומי (localhost בלבד):
השתמש בהגדרות ברירת המחדל.

#### לשימוש ברשת (גישה ממחשבים אחרים):

**צעד 1: גלה את כתובת ה-IP שלך**

Windows:
```bash
ipconfig
```
חפש את "IPv4 Address" (למשל: 192.168.1.100)

Mac/Linux:
```bash
ifconfig
# או
ip addr show
```

**צעד 2: צור קובץ `.env`**
```bash
cp .env.example .env
```

**צעד 3: ערוך את הקובץ `.env`**
```
VITE_SERVER_URL=http://192.168.1.100:5000
```
(החלף 192.168.1.100 בכתובת ה-IP שלך)

### 3. ודא שהשרת רץ
```bash
cd ../srv
./start_python.sh   # Linux/Mac
# או
start_python.bat    # Windows
```

השרת צריך לרוץ על `http://0.0.0.0:5000` (מאזין על כל הממשקים)

### 4. הפעל את הסימולטור
```bash
cd sim
npm run dev
```

הסימולטור יפתח על: `http://localhost:3001`

**לגישה ממחשבים אחרים ברשת:**
- פתח את הסימולטור מהדפדפן: `http://192.168.1.100:3001` (החלף בכתובת ה-IP שלך)
- ודא שחומת האש מאפשרת גישה לפורט 3001 ו-5000

### פעולות

#### 1. התחל משחק חדש
לחץ על **"🚀 התחל משחק חדש"**
- יוצר Game PIN חדש
- מאתחל את השרת
- מוצג ב-Game Info למעלה

#### 2. חבר משתתף
לחץ על **"🎮 הצטרף"** ליד שם המשתתף
- שולח בקשת POST ל-`/?join`
- משתתף מתווסף למשחק
- הכרטיס הופך לירוק
- מוצג **"🟢 מחובר"**

#### 3. נתק משתתף
לחץ על **"🚪 התנתק"** ליד משתתף מחובר
- שולח בקשת POST ל-`/?leave`
- משתתף מוסר מהמשחק
- הכרטיס חוזר למצב רגיל

## 🔌 API Calls

### חיבור משתתף
```javascript
POST http://localhost:5000/?join
Content-Type: application/json

{
  "user_id": "p1",
  "name": "אלכס כהן"
}
```

**תגובה:**
```json
{
  "status": "success",
  "message": "User joined successfully",
  "game_id": "123456",
  "user_count": 1
}
```

---

### ניתוק משתתף
```javascript
POST http://localhost:5000/?leave
Content-Type: application/json

{
  "user_id": "p1"
}
```

**תגובה:**
```json
{
  "status": "success",
  "message": "User left successfully",
  "user_count": 0
}
```

---

### אתחול משחק
```javascript
GET http://localhost:5000/?init
```

**תגובה:**
```json
{
  "status": "success",
  "message": "Quiz initialized",
  "game_id": "123456"
}
```

## 📡 WebSocket Events

### חיבור
```javascript
const socket = io('http://localhost:5000')

socket.on('connect', () => {
  console.log('✅ WebSocket connected')
})
```

### עדכון מספר משתתפים
```javascript
socket.on('user_count', (data) => {
  console.log('משתתפים:', data.count)
})
```

**Data:**
```json
{
  "count": 3
}
```

### אתחול משחק
```javascript
socket.on('game_initialized', (data) => {
  console.log('Game PIN:', data.game_id)
})
```

**Data:**
```json
{
  "game_id": "123456"
}
```
